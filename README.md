# TeamCreate

Repositorio para definir y ejecutar un **equipo de agentes** reusable.

## Estructura
- `team/roles/`: definición de roles del equipo.
- `team/workflows/`: flujos de colaboración.
- `team/templates/`: plantillas de trabajo.
- `.claude/agents/`: agente base para usar con Claude Code.

## Flujo recomendado
1. Abrir una tarea usando `team/templates/tarea.md`.
2. El orquestador define plan y asignación.
3. Arquitecto diseña.
4. Implementador ejecuta.
5. Revisor valida.
6. Orquestador cierra.

## Primera tarea de prueba
Usa este brief:

> Diseñar un pipeline simple que reciba un video, valide calidad básica y devuelva feedback estructurado.

Éxito mínimo:
- Documento de diseño.
- Plan de implementación en 3 fases.
- Checklist de validación.
